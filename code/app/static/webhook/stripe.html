3. วิธีลงทะเบียนใน Stripe

  1. เปิด Stripe Dashboard → Webhooks
  2. กด Add endpoint
  3. ใน Endpoint URL วาง URL จากช่องด้านบน (Copy URL)
  4. เลือก Events อย่างน้อย payment_intent.succeeded
  5. กด Add endpoint แล้วไปที่หน้า endpoint → Signing secret → Reveal → copy ค่า whsec_...
  6. นำค่า whsec_... ไปใส่ใน config ของ backend (STRIPE_WEBHOOK_SECRET)
