# คู่มือการใช้งานระบบ Food Court Management System

## สำหรับลูกค้า

### วิธีแลก Food Court ID

1. **ไปที่ Counter**
   - นำเงินสดหรือเลือกวิธีชำระเงินอื่น
   - แจ้งจำนวนเงินที่ต้องการแลก

2. **รับ Food Court ID**
   - พนักงานจะมอบ Food Court ID และ QR Code
   - เก็บ Food Court ID ไว้ให้ดี

3. **ใช้ Food Court ID**
   - นำ Food Court ID ไปใช้ที่ร้านค้า
   - สแกน QR Code หรือแจ้ง ID ให้พนักงานร้าน

4. **ตรวจสอบยอดเงิน**
   - สแกน QR Code ที่บัตรหรือใบเสร็จ
   - หรือเข้าเว็บไซต์และกรอก Food Court ID

5. **ขอคืนเงินที่เหลือ**
   - ไปที่ Counter
   - แจ้ง Food Court ID
   - รับเงินคืนตามวิธีที่เลือก

---

## สำหรับพนักงาน Counter

### การแลก Food Court ID

1. เปิดระบบ Counter
2. เลือกวิธีชำระเงินที่ลูกค้าเลือก
3. กรอกจำนวนเงิน
4. กด "ยืนยัน"
5. มอบ Food Court ID และ QR Code ให้ลูกค้า

### การคืนเงิน

1. เปิดระบบ Counter
2. สแกน QR Code หรือกรอก Food Court ID
3. ตรวจสอบยอดเงินคงเหลือ
4. คืนเงินตามวิธีที่ลูกค้าเลือก
5. ยืนยันการคืนเงิน

---

## สำหรับร้านค้า

### การรับ Food Court ID

1. เปิดระบบ POS
2. ใส่รายการอาหาร
3. สแกน QR Code หรือกรอก Food Court ID
4. ระบบจะหักยอดเงินอัตโนมัติ
5. มอบอาหารให้ลูกค้า

### การดูรายงาน

1. เข้าสู่ระบบรายงาน
2. เลือกช่วงวันที่
3. ดูสรุปยอดขาย
4. Export เป็น Excel (ถ้าต้องการ)

## Example Scenarios

### Scenario 1: ลูกค้าใหม่

**สถานการณ์:** ลูกค้าคนแรกที่มาใช้บริการ

**ขั้นตอน:**
1. ลูกค้าไปที่ Counter พร้อมเงินสด 500 บาท
2. พนักงาน Counter แลก Food Court ID ให้
3. ลูกค้าได้รับ: `FC-20241201-00001`
4. ลูกค้าไปสั่งอาหาร 150 บาท
5. ยอดเงินคงเหลือ: 350 บาท
6. ลูกค้าไปสั่งเครื่องดื่ม 50 บาท
7. ยอดเงินคงเหลือ: 300 บาท
8. ลูกค้าไปที่ Counter เพื่อขอคืนเงิน
9. พนักงานคืนเงิน 300 บาท

**ผลลัพธ์:**
- ลูกค้าใช้เงิน 200 บาท
- ได้รับเงินคืน 300 บาท
- มีบันทึก Transaction ทั้งหมด

---

### Scenario 2: ลูกค้าใช้หลายร้าน

**สถานการณ์:** ลูกค้าแลก 2,000 บาท และใช้หลายร้าน

**ขั้นตอน:**
1. แลก Food Court ID: 2,000 บาท
2. ร้าน A: ใช้ 350 บาท (เหลือ 1,650 บาท)
3. ร้าน B: ใช้ 280 บาท (เหลือ 1,370 บาท)
4. ร้าน C: ใช้ 150 บาท (เหลือ 1,220 บาท)
5. ร้าน D: ใช้ 420 บาท (เหลือ 800 บาท)
6. คืนเงินที่เหลือ: 800 บาท

**ผลลัพธ์:**
- ใช้เงินรวม: 1,200 บาท
- คืนเงิน: 800 บาท
- มี Transaction 4 รายการ

---

### Scenario 3: ลูกค้าใช้ LINE Pay

**สถานการณ์:** ลูกค้าแลกด้วย LINE Pay

**ขั้นตอน:**
1. ลูกค้าเลือกชำระด้วย LINE Pay
2. สแกน QR Code ด้วย LINE App
3. ชำระเงิน 1,500 บาท
4. ได้รับ Food Court ID
5. ใช้ที่ร้านต่างๆ
6. ขอคืนเงินที่เหลือผ่าน LINE Pay

**ผลลัพธ์:**
- ชำระเงินผ่าน LINE Pay
- คืนเงินผ่าน LINE Pay
- ไม่ต้องใช้เงินสด

## Tips & Best Practices

### สำหรับลูกค้า
- เก็บ Food Court ID ไว้ให้ดี
- ตรวจสอบยอดเงินก่อนใช้
- ขอคืนเงินที่เหลือก่อนออกจาก Food Court

### สำหรับพนักงาน
- ตรวจสอบยอดเงินก่อนคืนเงิน
- ยืนยันการคืนเงินทุกครั้ง
- แจ้งให้ลูกค้าทราบยอดเงินคงเหลือ

### สำหรับร้านค้า
- ตรวจสอบยอดเงินก่อนมอบอาหาร
- บันทึก Transaction ทุกครั้ง
- ตรวจสอบรายงานประจำวัน

## FAQ

**Q: Food Court ID หมดอายุหรือไม่?**
A: ไม่หมดอายุ แต่ควรใช้ภายในวันเดียวกัน

**Q: สามารถโอน Food Court ID ให้คนอื่นได้หรือไม่?**
A: ได้ แต่แนะนำให้ใช้เองเพื่อความปลอดภัย

**Q: ถ้า Food Court ID หายทำอย่างไร?**
A: ติดต่อ Counter พร้อมหลักฐานการแลก เพื่อขอคืนเงิน

**Q: สามารถใช้ Food Court ID ที่ร้านอื่นได้หรือไม่?**
A: ได้ ใช้ได้ทุกร้านใน Food Court
